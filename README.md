dot
===

dot files
override some config from [yadr](https://github.com/skwp/dotfiles)

See `SETUP.md` for unified environment setup (`zprezto + zsh + mise`).

Core layout:
- `config/`: editor/shell/git/tmux/lazygit managed config files
- `scripts/`: setup/cleanup/verify implementation and helper scripts
- `setup.sh`, `cleanup.sh`, `verify.sh`: root entrypoint wrappers
